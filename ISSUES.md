# Issues — Crosscue

Issues are tracked in **GitHub Issues**: https://github.com/AtomicTrxn/crossque/issues

Labels in use:

| Label | Meaning |
|-------|---------|
| `bug` | Something is broken |
| `enhancement` | New feature or improvement |
| `idea` | Exploratory / not yet scoped |
| `deferred` | Post-MVP; not in any active sprint |
| `area: solve` | Solve screen / gameplay |
| `area: ui` | UI / UX / layout |
| `area: parser` | Import parsers (.puz, .ipuz, .jpz) |
| `area: settings` | Settings screen |
| `area: android` | Android-specific |
| `area: infrastructure` | Core / DB / architecture / services |
| `area: stats` | Stats, streaks, sessions |

Historical closed issues (shipped before GitHub tracking began) are documented in [COMPLETED_SPRINTS.md](COMPLETED_SPRINTS.md).
